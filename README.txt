- Instalar o banco de dados instalando o MYSQL Installer, disponível em: https://dev.mysql.com/downloads/installer/.

- A versão atual utilizada neste projeto é a versão 5.7.12 do instalador.

- É necessário instalar:
	-> MYSQL Server
	-> MYSQL Workbench (ferramenta visual para gerenciar o banco de dados)
	-> MYSQL for Visual Studio (extensão para utilizar junto da IDE)
	-> MYSQL Connector (plugin necessário para a conexão correta do banco)

- As versões dos produtos acima já vem atualizados no momento da instalação, não sendo necessário a asserção dos mesmos.
